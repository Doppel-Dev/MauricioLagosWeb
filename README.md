# Mauricio Lagos Web Portfolio

This repository contains the source code for Mauricio Lagos' professional portfolio, featuring a modern web application built with a .NET 10 Web API backend and a React (Vite) frontend.

## Project Structure

- **src/MauricioLagos.Api**: ASP.NET Core Web API handling the backend logic and data.
- **src/MauricioLagos.Client**: React frontend built with TypeScript, Vite, and Tailwind CSS.
- **CV**: Contains professional curriculum vitae.

## Tech Stack

### Backend
- .NET 10
- ASP.NET Core Web API
- C#

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- PostCSS

## Getting Started

### Prerequisites
- .NET 10 SDK
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Doppel-Dev/MauricioLagosWeb.git
   ```

2. **Backend Setup**:
   ```bash
   cd src/MauricioLagos.Api
   dotnet restore
   dotnet run
   ```

3. **Frontend Setup**:
   ```bash
   cd src/MauricioLagos.Client
   npm install
   npm run dev
   ```
