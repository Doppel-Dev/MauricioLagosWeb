# Mauricio Lagos - Developer Portfolio

Este es un portafolio profesional desarrollado con tecnologías modernas, demostrando capacidades de desarrollo Full Stack.

## Tecnologías Utilizadas

- **Backend:** .NET 10 Web API (C#)
- **Frontend:** React + TypeScript + Vite
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Iconos:** Lucide React

## Estructura del Proyecto

- `src/MauricioLagos.Api/`: Backend que sirve la información profesional a través de una API REST.
- `src/MauricioLagos.Client/`: Frontend interactivo y animado que consume la API.

## Cómo Ejecutar el Proyecto

### 1. Ejecutar el Backend (.NET)

Desde la raíz del proyecto o en la carpeta del backend:
```bash
dotnet run --project src/MauricioLagos.Api/MauricioLagos.Api.csproj
```
La API estará disponible en `http://localhost:5000` (o el puerto configurado por defecto).

### 2. Ejecutar el Frontend (React)

Desde la carpeta `src/MauricioLagos.Client/`:
```bash
npm run dev
```
El sitio estará disponible en `http://localhost:5173`.

## Características Destacadas

- **Diseño Responsive:** Adaptable a móviles, tablets y escritorio.
- **Animaciones Fluidas:** Uso de Framer Motion para entradas suaves y transiciones atractivas.
- **Arquitectura Limpia:** Separación clara entre Backend y Frontend.
- **Placeholder para Fotos:** Espacios diseñados para incluir fotos profesionales fácilmente.
